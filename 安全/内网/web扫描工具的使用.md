# 漏洞测试报告
- 漏洞名称：具体类型
- 风险等级：高/中/低
- 漏洞描述：概念/原理/原因
- 漏洞详情：url/参数/payload/判断依据/请求响应包
- 漏洞危害：描述危害
- 修复建议：修补的建议
- 参考资料：参考连接CVE编号/厂家漏洞说明
